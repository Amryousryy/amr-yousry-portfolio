## OPERATIONAL DIRECTIVES

### Primary Focus

The platform's primary focus is now:
1. **Product Development** — Building products with the platform
2. **Operational Monitoring** — Observing health metrics
3. **Evidence Collection** — Gathering engineering evidence
4. **Release Governance** — Managing version-based evolution

### What's Prohibited

The following are no longer permitted without governance approval:
1. **Platform Extension** — Adding new primitives without evidence
2. **Architecture Changes** — Modifying the 5-layer architecture
3. **Contract Violations** — Breaking frozen contracts
4. **Behavior Redefinition** — Implementing behavior outside the Behavior API

### What's Encouraged

The following are encouraged:
1. **Product Building** — Using the platform to build real products
2. **Evidence Collection** — Documenting friction points and metrics
3. **Developer Feedback** — Providing feedback on platform usability
4. **Compliance Monitoring** — Running validation and health checks
